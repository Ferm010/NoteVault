## Структура расходов проекта

```
Себестоимость = Прямые расходы + Накладные расходы
Прямые расходы = ФОТ + Страховые взносы
Накладные расходы = Услуги связи + Коммунальные + Реклама + Прочие
```

---

## 1. Прямые расходы

### 1.1 Фонд оплаты труда (ФОТ)

**Формула:**
```
ФОТ_год = ЗП × N
```
- `ЗП` — заработная плата в месяц, руб.
- `N` — количество месяцев в году (12 мес.)

**Пример:** Программист получает 25 000 руб./мес.
```
ФОТ_год = 25 000 × 12 = 300 000 руб.
```

### 1.2 Стоимость трудозатрат за 1 час

**Формула:**
```
Сt_час = ФОТ_год / N_рв
```
- `N_рн` — норма рабочего времени при 40-часовой неделе (на 2026 год = **1972 часа**)

**Пример:**
```
Сt_час = 300 000 / 1972 = 151,59 руб./час
```

> [!NOTE] Норму рабочего времени проверяйте на garant.ru/calendar/buhpravo/ для текущего года.

### 1.3 Таблица трудозатрат

Заполните таблицу с разбивкой по видам работ:

| Вид работы | Трудозатраты (час) | Стоимость (руб.) |
|------------|-------------------|------------------|
| Анализ требований | | |
| Проектирование | | |
| Разработка | | |
| Тестирование | | |
| **Итого** | | |

```
Стоимость по каждой строке = Трудозатраты × Сt_час
```

### 1.4 Страховые взносы

**Тариф:** 30% (основной) + 0,2% (несчастные случаи) = **30,2%**

**Формула:**
```
СВ = РОТ × ТСВ / 100%
```
- `РОТ` — расходы на оплату труда (ФОТ), руб.
- `ТСВ` — тариф страховых взносов (%)

**Пример:**
```
СВ = 300 000 × 30,2% = 90 600 руб.
```

## 2. Накладные расходы

### 2.1 Временные сроки выполнения проекта

**Формула:**
```
СП = t_общ / 8
```
- `t_общ` — общая сумма трудозатрат из таблицы выше, час.
- `8` — стандартный рабочий день (часы)

### 2.2 Услуги связи (интернет + телефон)

**Формула:**
```
Ру.с. = (Р_интернет + Р_телефон + ...) / Кол-во_дней_мес × СП
```
- `Кол-во_дней_мес` — среднее количество рабочих дней в месяце (~21)

**Пример:** Интернет 550 руб./мес. + телефон 400 руб./мес., проект = 40 дней
```
Ру.с. = (550 + 400) / 21 × 40 = 1 809,52 руб.
```

### 2.3 Коммунальные услуги

**Формула:**
```
Р_коммунальные = n_к.у. × S
```
- `n_к.у.` — средняя ставка за 1 кв.м. (110–120 руб.)
- `S` — площадь помещения, кв.м.

**Пример:** 110 руб./кв.м. × 18 кв.м. = 1 980 руб.

### 2.4 Реклама

**Формула:**
```
Р_реклама = РОТ × n_реклама / 100%
```
- `n_реклама` — норматив (рекомендуется **15%**, диапазон 10–20%)

**Пример:**
```
Р_реклама = 300 000 × 15% = 45 000 руб.
```

### 2.5 Прочие расходы

**Формула:**
```
Р_прочие = РОТ × n_прочие / 100%
```
- `n_прочие` — норматив (рекомендуется **10%**)

К прочим относится: аренда сервера, лицензии ПО и т.д.

**Пример:**
```
Р_прочие = 300 000 × 10% = 30 000 руб.
```

### 2.6 Итого накладные расходы

```
Р_накладные = Ру.с. + Р_коммунальные + Р_реклама + Р_прочие
```

## 4. Стоимость владения (Св)

Включает эксплуатационные расходы и затраты на функционирование:

```
Св = Эр + Зф
```
- `Эр` — сумма эксплуатационных расходов (часы × Сt_час = 151,59 руб./час)
- `Зф` — сумма затрат на функционирование (сервер, хостинг, домен и т.д.)

> [!TIP] На практике затраты на владение в первый год не должны превышать стоимость разработки.

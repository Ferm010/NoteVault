
Подсчет суммы для уникальный ID

```SQL
SELECT SUM(price) FROM price_table
GROUP BY user_id
```

- **`COUNT()`**: Считает количество строк или ненулевых значений в столбце (`COUNT(*)`, `COUNT(col)`).
- **`SUM()`**: Вычисляет сумму числовых значений в столбце.
- **`AVG()`**: Определяет среднее арифметическое значение, игнорируя `NULL`.
- **`MIN()`**: Находит наименьшее значение.
- **`MAX()`**: Находит наибольшее значение.

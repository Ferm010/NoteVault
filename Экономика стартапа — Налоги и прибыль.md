## Упрощённая система налогообложения (УСН)

Согласно ст. 346.12 НК РФ, налогоплательщики — организации и ИП на УСН.

### Объект налогообложения

| Вариант | Ставка | Диапазон (субъекты РФ) |
|---------|--------|----------------------|
| **Доходы** | **6%** | 1–6% |
| **Доходы минус расходы** | **15%** | 5–15% |

### Какой вариант выбрать для IT-стартапа?

IT-сфера не является материалоёмкой — удельный вес расходов в выручке обычно **менее 60%**. Поэтому целесообразно выбирать **«Доходы»** со ставкой **6%**.

## Расчёт единого налога (УСН)

### 1. Начисленный налог

```
УСН_нач = Д × СУСН / 100%
```
- `Д` — доход (выручка), руб.
- `СУСН` — ставка налога (%)

**Пример:** Доход = 500 000 руб., ставка = 6%
```
УСН_нач = 500 000 × 6% = 30 000 руб.
```

### 2. Уменьшение налога на страховые взносы

Налогоплательщики с объектом «Доходы» уменьшают сумму налога на страховые взносы, но **не более чем на 50%**.

### 3. Минимальный налог

```
УСН_min = Д × 1%
```

### 4. Налог к уплате в бюджет

Одна из формул (в зависимости от ситуации):

```
Вариант А: УСН_бюджет = УСН_нач - УСН_min
Вариант Б: УСН_бюджет = УСН_нач - Страховые_взносы
```

## Расчёт цены продукта (выручки)

### Формула выручки

```
В_реал = Ср + П
```
- `Ср` — себестоимость работ (затраты на создание ПО)
- `П` — планируемая прибыль

### Расчёт прибыли

```
П = Ср × НR / 100%
```
- `НR` — уровень рентабельности проекта (**от 10% до 30%**, на выбор)

> [!WARNING] Чем больше % рентабельности, тем выше прибыль, но ниже конкурентоспособность продукта.

### Пример расчёта цены

| Параметр | Значение |
|----------|----------|
| Себестоимость (Ср) | 500 000 руб. |
| Рентабельность (НР) | 20% |
| Прибыль (П) | 100 000 руб. |
| **Выручка (В_реал)** | **600 000 руб.** |

### Расчёт налога и чистой прибыли

```
УСН_нач = 600 000 × 6% = 36 000 руб.
УСН_бюджет = УСН_нач - Страховые_взносы (если взносы > 0)

П_чистая = П - УСН_бюджет
```

## Расчёт точки безубыточности и окупаемости

### Доход в месяц

```
Р_доход = Средний_чек × Целевой_рынок × 12 мес.
```

### Окупаемость

Определите, за какой срок проект выйдет на точку безубыточности:

```
Срок_окупаемости = Смета_затрат / (Доход_в_месяц - Расходы_в_месяц)
```

> [!TIP] Точка безубыточности — момент, когда cumulative доход равен cumulative расходам. После этой точки проект начинает приносить прибыль.

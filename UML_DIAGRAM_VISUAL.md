## Матрица зависимостей Use Cases

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    ЗАВИСИМОСТИ МЕЖДУ USE CASES                              │
└─────────────────────────────────────────────────────────────────────────────┘

UC-001 (Регистрация)
  └─→ UC-002 (Вход) - после успешной регистрации
      └─→ UC-003 (Поиск модов) - после входа
          ├─→ UC-004 (Просмотр деталей) - выбрать мод
          │   └─→ UC-005 (Избранное) - добавить в избранное
          │
          └─→ UC-006 (Создать модпак) - нажать "Создать"
              ├─→ UC-007 (Добавить моды) - добавить моды
              │   └─→ UC-008 (Шаблон) - сохранить как шаблон
              │
              ├─→ UC-009 (Экспорт локально) - экспортировать
              │
              └─→ UC-010 (Экспорт в Drive) - экспортировать

UC-011 (Профиль) - доступен с любого экрана
  ├─→ UC-012 (Язык) - выбрать язык
  ├─→ UC-013 (Тема) - выбрать тему
  ├─→ UC-014 (Поддержка) - техническая поддержка
  └─→ UC-015 (Выход) - выход из аккаунта
      └─→ UC-001 (Регистрация) - вернуться на экран входа
```
